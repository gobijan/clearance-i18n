# Clearance I18n

Adds multilingual support to [clearance] using YML locale files.

[clearance]: https://github.com/thoughtbot/clearance

Read [CONTRIBUTING.md](/CONTRIBUTING.md) to contribute.

## Install

Include the gem in your Gemfile:

```ruby
gem 'clearance-i18n'
```

Bundle:

    bundle
